# 千千的完整技能配置

## 两个核心技能

### 1️⃣ 图片生成视频 - flash_longxia (帧龙虾)

**位置**: `/Users/mima0000/.openclaw/workspace/openclaw_upload/flash_longxia/`

**功能**: 使用帧龙虾 API 将图片转换成视频

**使用方法**:
```bash
cd /Users/mima0000/.openclaw/workspace/openclaw_upload/flash_longxia
python3 zhenlongxia_workflow.py <图片路径>
```

**流程**:
1. 上传本地图片 → 得到 OSS URL
2. 图生文 imageToText → 得到系统提示词
3. 发起 generateVideo → 得到任务 id
4. 轮询 getById(id) → 等待视频完成（每 30 秒，最多 30 分钟）
5. 下载 MP4 到 output 目录

**Token**: 已保存在 `token.txt`

---

### 2️⃣ 多平台视频发布 - multi-platform-upload

**位置**: `/Users/mima0000/.openclaw/workspace/openclaw_upload/`

**功能**: 上传视频到四个平台

**支持平台**:
- 抖音 (douyin)
- 快手 (kuaishou)
- 视频号 (shipinhao)
- 小红书 (xiaohongshu)

**使用方法**:
```bash
cd /Users/mima0000/.openclaw/workspace/openclaw_upload
.venv313/bin/python upload.py -p <平台> "<视频路径>" "<标题>" "<文案>" "<标签>"
```

**登录状态**: 已保存在 `cookies/<平台>/` 目录

## 🎬 完整工作流

当千千说「生成视频并上传」或发送图片时：

### 步骤 1: 保存图片
```bash
# 图片自动保存到
/Users/mima0000/.openclaw/workspace/inbound_images/<图片文件名>
```

### 步骤 2: 生成视频
```bash
cd /Users/mima0000/.openclaw/workspace/openclaw_upload/flash_longxia
python3 zhenlongxia_workflow.py "/Users/mima0000/.openclaw/workspace/inbound_images/<图片文件名>"
```

**输出**: `/Users/mima0000/.openclaw/workspace/openclaw_upload/flash_longxia/output/video.mp4`

### 步骤 3: 发布到所有平台
```bash

.venv313/bin/python upload.py -p shipinhao "output/video.mp4" "" "" "可爱，日常"

# 抖音
.venv313/bin/python upload.py -p douyin "output/video.mp4" "可爱日常" "可可爱爱的一天" "可爱，日常"

# 快手
.venv313/bin/python upload.py -p kuaishou "output/video.mp4" "可爱日常" "可可爱爱的一天" "可爱，日常"

# 小红书
.venv313/bin/python upload.py -p xiaohongshu "output/video.mp4" "可爱日常" "可可爱爱的一天" "可爱，日常"
```
